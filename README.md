# UniTrade

Campus marketplace mobile application for SDG.

## Project Structure

This is a native Android application built with Java.

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator or device

## Firebase Configuration

1. Download `google-services.json` from Firebase Console
2. Place it in `app/` folder
3. Go to project setting ->  service account -> generate new private key(Java) -> rename it to service-account.json
4. Place it in /app/src/main/assets
5. Sync Gradle

## Requirements

- Android Studio Arctic Fox or later
- JDK 11 or later
- Android SDK 24+
